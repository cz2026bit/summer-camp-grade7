# 🚀 暑假冲刺营 · 小升初 40 天在线作业

为六年级暑假(即将升初一)的学生设计的在线暑假作业系统。纯静态网页,无需安装任何软件,浏览器打开即用。

## ✨ 特色

- **边学边练**:每学完一个知识点,马上做一道对应的题,提交后立刻显示对错和讲解
- **四科齐全**:数学 + 英语 + 语文 + 副科(历史/地理/生物/道法轮换),每天约 2~4 小时
- **动画教学**:数轴动画、温度计动画、单词翻牌卡、口诀逐行出现等互动效果,让学习更有趣
- **循序渐进**:40 天分 6 个阶段,由易到难(六年级复习 → 有理数 → 代数式 → 方程 → 综合 → 模拟测评)
- **自动统分**:全部做完后统计总分和分科成绩,展示全部题目的对错和正确答案
- **图片报告**:一键导出 PNG 图片报告(含分数、分科表、全部题目对错、错题答案、家长签名栏),微信直接发给家长
- **进度保存**:学习进度和成绩自动保存在浏览器本地,可自由选择学习任意一天

## 📖 使用方法

**在线使用**:直接访问 GitHub Pages 页面即可。

**本地使用**:下载本仓库,双击打开 `index.html`。

## 📅 40 天课程安排

| 阶段 | 天数 | 数学 | 英语 | 语文 |
|------|------|------|------|------|
| 一 · 温故知新 | 1-7 | 分数复习、负数、数轴 | 校园词汇、Be动词、代词、There be | 字音字形、成语、病句、古诗预习 |
| 二 · 有理数运算 | 8-14 | 有理数四则运算、乘方 | 一般现在时、疑问词、可数名词 | 记叙文阅读、《世说新语》、修辞 |
| 三 · 代数式初步 | 15-21 | 用字母表示数、合并同类项 | 现在进行时、情态动词、介词 | 《论语》、人物描写、写作审题 |
| 四 · 一元一次方程 | 22-28 | 解方程、方程应用题 | 一般过去时、阅读理解专项 | 环境描写、《诫子书》、说明文 |
| 五 · 综合提升 | 29-35 | 方程应用、几何初步、找规律 | 将来时、完形填空、书面表达 | 议论文、综合性学习、名著精读 |
| 六 · 模拟测评 | 36-40 | 综合复习 + 模拟测试 | 综合复习 + 模拟测试 | 综合复习 + 模拟测试 |

副科(历史/地理/生物/道法)每天轮换一门,每周第 7 天为小测复习日。

## 🗂️ 项目结构

```
index.html          # 入口页面
css/style.css       # 全部样式与动画
js/app.js           # 应用逻辑(渲染、判分、报告导出)
js/data/curriculum.js  # 40 天课程大纲
js/data/dayXX.js    # 每天的学习内容与题目(逐天添加)
```

## ➕ 如何添加新的一天

1. 复制 `js/data/day01.js` 为 `js/data/day02.js`,修改 `window.DAYS[2] = {...}` 中的内容
2. 在 `index.html` 中添加 `<script src="js/data/day02.js"></script>`
3. 完成!首页对应天数的卡片会自动解锁
