# 🚀 暑假冲刺营 · 小升初 40 天在线作业

为六年级暑假(即将升初一)的学生设计的在线暑假作业系统。纯静态网页,无需安装任何软件,浏览器打开即用。

## ✨ 特色

- **每天一页**:学习 📖 → 做题 ✏️ → 对答案 ✅ → 打分 🏆 → 导出报告 📤 的完整闭环
- **数学 + 英语**:每天约 2~4 小时学习量,小升初衔接内容
- **动画教学**:数轴动画、温度计动画、单词翻牌卡、口诀逐行出现等互动效果,让学习更有趣
- **循序渐进**:40 天分 6 个阶段,由易到难(六年级复习 → 有理数 → 代数式 → 方程 → 综合 → 模拟测评)
- **自动打分**:交卷后自动批改、逐题讲解,成绩圆环动画展示
- **家长报告**:一键导出 HTML 学习报告(含错题清单和讲解),微信发给家长即可查看/打印
- **进度保存**:学习进度和成绩自动保存在浏览器本地,可自由选择学习任意一天

## 📖 使用方法

**在线使用**:直接访问 GitHub Pages 页面即可。

**本地使用**:下载本仓库,双击打开 `index.html`。

## 📅 40 天课程安排

| 阶段 | 天数 | 数学 | 英语 |
|------|------|------|------|
| 一 · 温故知新 | 1-7 | 分数复习、负数、数轴、有理数入门 | 校园词汇、Be动词、代词、名词、There be |
| 二 · 有理数运算 | 8-14 | 有理数四则运算、乘方、科学记数法 | 一般现在时、疑问词、可数名词 |
| 三 · 代数式初步 | 15-21 | 用字母表示数、合并同类项、去括号 | 现在进行时、情态动词、介词 |
| 四 · 一元一次方程 | 22-28 | 解方程、方程应用题 | 一般过去时、阅读理解专项 |
| 五 · 综合提升 | 29-35 | 方程应用、几何初步、找规律 | 将来时、完形填空、书面表达 |
| 六 · 模拟测评 | 36-40 | 综合复习 + 模拟测试 | 综合复习 + 模拟测试 |

每周第 7 天为小测复习日,巩固一周所学。

## 🗂️ 项目结构

```
index.html          # 入口页面
css/style.css       # 全部样式与动画
js/app.js           # 应用逻辑(渲染、判分、报告导出)
js/data/curriculum.js  # 40 天课程大纲
js/data/dayXX.js    # 每天的学习内容与题目(逐天添加)
```

## ➕ 如何添加新的一天

1. 复制 `js/data/day01.js` 为 `js/data/day02.js`,修改 `window.DAYS[2] = {...}` 中的内容
2. 在 `index.html` 中添加 `<script src="js/data/day02.js"></script>`
3. 完成!首页对应天数的卡片会自动解锁
